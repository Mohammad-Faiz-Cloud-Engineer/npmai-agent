# NPMAI-Agent-Suite
npmai-agent (internally known as the npmai-agent-suite) is a production-grade AI agent framework built on top of the NPMAI ECOSYSTEM.It gives any Python developer a fully autonomous, multi-LLM agentic pipeline with 21 integrated tool classes,orchestrated by a four-role LLM pipeline (Planner → Coder → Auditor → Verifier).
